# PatchWell
PatchWell is a dependency vulnerability assessment tool that analyzes project manifest files or public GitHub repositories against the OSV and GitHub Advisory databases to identify known security vulnerabilities. The platform employs a large language model to translate raw vulnerability data into plain-language risk explanations, contextual severity scoring, patch guidance with breaking change analysis and remediation recommendations for unmaintained or unpatchable dependencies. Findings are consolidated into a risk-ranked executive report designed to support rapid remediation planning.
